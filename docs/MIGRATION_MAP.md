# Migration Map

Archived `gear-*` repositories now map into this consolidated repository.

## Module families

- Sports trackers -> `examples/sports/` + `src/adapters/sports_*`
- Policy and status watchers -> `examples/policy/` + `src/adapters/policy_*`
- Monitoring and alert workflows -> `examples/ops/` + `src/notifiers/*`

## Note

Each migrated implementation should keep a short changelog note describing parity gaps and improvements.
